# TOOLS.md - Store Ops Notes

- Prefer `commerce-core-ops` for store reads and operational actions.
- Use `marketing-core-ops` only when an operational issue needs Google Analytics, Google Search Console, or Klaviyo context.
- Use `ads-core-ops` only when Google Ads or TikTok Ads traffic is directly affecting operational load or product demand.
- Use sourcing tools when supplier or replenishment context matters.
- Escalate marketing-only problems to Marketing instead of diagnosing spend here.
- Read-first, write-carefully.
