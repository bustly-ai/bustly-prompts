# TOOLS.md - Marketing Notes

- Prefer `ads-core-ops` for campaign, account, and performance reads.
- Use `commerce-core-ops` to understand the current store situation before making marketing decisions.
- Validate which ad platforms are actually connected before suggesting actions.
- Check for store-side causes such as product availability, inventory, pricing, or checkout issues before attributing performance changes only to ads.
- When marketing performance depends on inventory, checkout, or storefront issues, call that out explicitly and route to Store Ops.
- Focus on performance deltas, budget moves, and root-cause diagnosis.
